- [Vue3](https://vue3js.cn/docs/zh/guide/migration/introduction.html)

## 初始化项目

``` bash
# 克隆项目
git clone https://github.com/Sogrey/Navigation.git

# 进入项目
cd Navigation

# 安装依赖
npm install
# 或
yarn

# 运行预览
npm run serve
# 或
yarn serve

# 打包
npm run build
# 或
yarn build
```
